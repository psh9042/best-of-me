Hypertext Markup Language (HTML) is the standard markup language for pages and web applications. Web browers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantic ally and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages.
